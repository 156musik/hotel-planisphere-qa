# Navigation Analysis

## 調査メモ

ログイン状態における全ページのナビゲーション仕様（テスト観点）
ページ分類の全体像
コードベース上に存在する ja/ 配下の全ページを3つのグループに分類できる。

グループ	ページ	分類根拠
A: 動的切替あり	index.html、plans.html	signup-holder 等のIDが存在し、setLoginNavbar() が呼ばれる
B: 静的固定	login.html、signup.html、mypage.html、icon.html	ナビ項目がHTMLに固定。JSによる切替なし
C: ナビリンクなし	reserve.html、confirm.html、about.html	ナビバー自体が存在しないか、ブランド名のみ
コード根拠: ja/index.html:29-42、ja/plans.html:30-43、src/index.js:5-8、src/plans.js:8-11

グループA：動的切替あり（index.html / plans.html）
初期HTML状態（JSで操作される前）

signup-holder  → d-block（会員登録 表示）
login-holder   → d-block（ログイン 表示）
mypage-holder  → d-none （マイページ 非表示）
logout-holder  → d-none （ログアウト 非表示）
コード根拠: ja/index.html:29-42

ログイン状態別の表示仕様
状態	会員登録	ログイン	マイページ	ログアウト
未ログイン	表示	表示	非表示	非表示
ログイン済み	非表示	非表示	表示	表示
コード根拠: src/lib/session.js:102-110（setLoginNavbar() の実装）

テスト観点（グループA共通）
未ログイン時

「会員登録」リンクが表示されていること
「ログイン」ボタンが表示されていること
「マイページ」リンクが表示されていないこと
「ログアウト」ボタンが表示されていないこと
ログイン済み時

「会員登録」リンクが表示されていないこと
「ログイン」ボタンが表示されていないこと
「マイページ」リンクが表示されていること
「ログアウト」ボタンが表示されていること
ログアウト操作

ログアウトボタン押下で logout() が実行されること（src/lib/session.js:107-109）
ログアウト後に index.html へ遷移すること（action="./index.html"）
ログアウト後、ナビが未ログイン状態の表示に戻っていること
グループB：静的固定（ページごとに個別）
login.html
ナビ項目	表示内容	ログイン状態による変化
ホーム	表示（固定）	なし
宿泊予約	表示（固定）	なし
会員登録	表示（固定）	なし
ログイン	表示・active（固定）	なし
アクセス制御: ログイン済みでアクセスすると redirectToTop() でトップページへリダイレクト（src/login.js:5-8）

テスト観点

未ログイン時: 4項目が固定表示されていること
ログイン済みでURLを直接入力した場合: ログインページが表示されず、トップページへリダイレクトされること
ナビの表示内容はログイン状態に関わらず変化しないこと
signup.html
ナビ項目	表示内容	ログイン状態による変化
ホーム	表示（固定）	なし
宿泊予約	表示（固定）	なし
会員登録	表示・active（固定）	なし
ログイン	表示（固定）	なし
アクセス制御: ログイン済みでアクセスすると redirectToTop() でトップページへリダイレクト（src/signup.js:5-8）

テスト観点

未ログイン時: 4項目が固定表示されていること
ログイン済みでURLを直接入力した場合: 会員登録ページが表示されず、トップページへリダイレクトされること
ナビの表示内容はログイン状態に関わらず変化しないこと
mypage.html
ナビ項目	表示内容	ログイン状態による変化
ホーム	表示（固定）	なし
宿泊予約	表示（固定）	なし
マイページ	表示・active（固定）	なし
ログアウト	表示（固定）	なし
アクセス制御: 未ログインでアクセスすると redirectToTop() でトップページへリダイレクト（src/mypage.js:13-16）

ログアウトバインド: setLoginNavbar() ではなく src/mypage.js:51-53 で個別バインド

テスト観点

未ログインでURLを直接入力した場合: マイページが表示されず、トップページへリダイレクトされること
ログイン済み時: マイページが表示され、ログアウトボタンが機能すること
ナビの表示内容はログイン状態に関わらず常に同じ4項目であること（切替処理がないため）
icon.html
ナビ項目	表示内容	ログイン状態による変化
ホーム	表示（固定）	なし
宿泊予約	表示（固定）	なし
マイページ	表示・active（固定）	なし
ログアウト	表示（固定）	なし
アクセス制御: 未ログインでアクセスすると redirectToTop() でトップページへリダイレクト（src/icon.js:5-8）。加えてプリセットユーザ（user.preset === true）もリダイレクト（src/icon.js:14-15）。

テスト観点

未ログインでURLを直接入力した場合: トップページへリダイレクトされること
プリセットユーザでログイン後にアクセスした場合: トップページへリダイレクトされること
会員登録ユーザ（localStorage保存ユーザ）でログイン済みの場合のみ: ページが表示されること

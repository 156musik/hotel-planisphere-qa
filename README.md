# Hotel Planisphere QA Portfolio

## Overview

本リポジトリは、ログイン機能に対するE2Eテスト自動化を目的としたサンプルプロジェクトです。
Playwrightを用いて、基本的な認証フローおよび入力バリデーションの検証を行っています。

---

## Target

検証対象はシンプルなログイン画面とし、以下をカバーしています。

* 正常ログイン
* 入力不備（メールアドレス・パスワード）
* 認証失敗時のエラーメッセージ表示

---

## Test Design

以下の観点でテストを設計しています。

* 入力バリデーション

  * 未入力
  * フォーマット不正（メールアドレス）
* 認証処理

  * 正常系（正しい認証情報）
  * 異常系（誤った認証情報）
* UIフィードバック

  * エラーメッセージの表示有無
  * 表示内容の確認

各テストケースは「1ケース1意図」を意識して分割しています。

---

## Test Cases

主なテストケース：

* 正常ログイン
* メールアドレス未入力
* パスワード未入力
* メールアドレス形式不正
* 認証失敗

---

## Implementation

* PlaywrightのCode Generatorを利用して初期スクリプトを生成
* 生成コードをベースに以下を調整

  * セレクタの安定性向上（roleベースを優先）
  * 可読性の改善
  * 不要な処理の削除

---

## Tech Stack

* Playwright
* TypeScript
* Node.js

---

## AI Assistance

本プロジェクトでは以下の用途でAIを補助的に利用しています。

* テスト観点の整理

* 不明点の技術調査

実装および最終的な設計判断は自身で行っています。
